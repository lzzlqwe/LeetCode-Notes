# 代表性题目

## [1.二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/description/)
思路：经典题目。整个树的深度 = max(左子树深度，右子树深度) + 1    
代码：
```
class Solution {
    public int maxDepth(TreeNode root) {
        if(root == null)
            return 0;
        return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
    }
}
```
